ACNL_PIC_Outputer

制作:Hack-Usagi
初期公開:2018/9/11 最終更新:2018/9/11
動作確認:Windows7 32bit/Windows10 64bit

とびだせ どうぶつの森 Ver.1.5(amiibo+)のプロフィール欄の画像を出力するツール

使い方

1.ACNL_PIC_Changer.exeを実行
2.「select save data」を押してセーブデータ(garden_plus.dat)を選択
3.「start」を押して差し替えスタート
4.保存ダイアログがでるので任意の場所に保存
